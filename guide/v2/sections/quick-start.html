شروع سریع

1. دریافت توکن

پس از خرید اکانت، یک توکن اختصاصی برای شما صادر می‌شود که برای استفاده از SDK ضروری است.

نکته: توکن را در محیط توسعه امن نگهداری کنید و هرگز آن را در کد front-end خود به صورت مستقیم قرار ندهید.

2. افزودن کتابخانه

فایل‌های مورد نیاز را به پروژه خود اضافه کنید:

<link rel="stylesheet" href="https://sdk.armogroup.tech/makeup/v2/makeup.min.css">
<script src="https://sdk.armogroup.tech/makeup/v2/makeup.js"></script>

3. راه‌اندازی اولیه

3.1. افزودن المان پایه

<div id="armo-makeup-view"></div>

3.2. راه‌اندازی SDK

document.addEventListener("DOMContentLoaded", () => {
    // تنظیم توکن
    window.Makeup.token = "YOUR_TOKEN_HERE";

    // ایجاد نمونه از Makeup
    const makeupApp = new window.Makeup({
        colors: [
            { code: "01", color: "#000000" },
            { code: "02", color: "#4B0082" },
            // استفاده از URL تصویر برای استخراج خودکار رنگ
            { code: "03", url: "https://example.com/lipstick-image.jpg" }
        ],
        pattern: "normal",
        face: "lips",
        transparency: 0.7, // تنظیم شفافیت کلی
        onReady: () => {
            console.log("SDK آماده استفاده است");
        },
        onError: (error) => {
            console.error("خطا در SDK:", error);
        }
    });

    window.makeupApp = makeupApp;
});

گزینه‌های پیکربندی

گزینه نوع پیش‌فرض توضیحات
colors Array [] آرایه‌ای از رنگ‌های قابل انتخاب
pattern String "normal" الگوی پیش‌فرض آرایش
face String "lips" نوع آرایش پیش‌فرض
showColorPicker Boolean true نمایش/عدم نمایش انتخابگر رنگ داخلی
transparency Number undefined تنظیم شفافیت کلی آرایش. عددی بین 0 تا 1. در صورت عدم تعیین، از مقادیر پیش‌فرض هر نوع آرایش استفاده می‌شود.
onReady Function () => {} تابع callback برای زمان آماده شدن SDK
onError Function () => {} تابع callback برای زمان بروز خطا

بررسی وضعیت SDK

می‌توانید وضعیت فعلی SDK را با استفاده از متد getStatus() بررسی کنید: 

const status = makeupApp.getStatus();
// status می‌تواند یکی از این مقادیر باشد:
// - 'initializing': در حال راه‌اندازی اولیه
// - 'loading': در حال بارگذاری منابع
// - 'ready': آماده استفاده
// - 'error': خطا در راه‌اندازی

4. استخراج خودکار رنگ از تصاویر

یکی از ویژگی‌های جدید SDK، قابلیت استخراج خودکار رنگ از تصاویر است. با این قابلیت می‌توانید:

  • به جای تعیین دستی رنگ، SDK رنگ غالب را از تصویر استخراج می‌کند
  • اگر هر دو مقدار color و url تعیین شوند، color اولویت دارد
  • در صورت خطا در استخراج رنگ، از رنگ خاکستری پیش‌فرض استفاده می‌شود

نکته مهم: تصاویر باید دسترسی CORS داشته باشند تا SDK بتواند آنها را پردازش کند.

4.1. تعریف رنگ‌ها با استفاده از URL تصویر

const makeupApp = new window.Makeup({
    colors: [
        // تعیین مستقیم رنگ
        { code: "01", color: "#FF0000" },
        
        // استخراج خودکار رنگ از تصویر
        { code: "02", url: "https://example.com/products/lipstick-red.jpg" },
        
        // اولویت با color است
        { code: "03", color: "#00FF00", url: "https://example.com/green-product.jpg" },
        
        // تعیین نوع آرایش برای رنگ
        { code: "04", url: "https://example.com/eyeshadow-blue.jpg" }
    ],
    // سایر تنظیمات...
});

4.2. افزودن رنگ جدید از URL تصویر

همچنین می‌توانید به صورت پویا رنگ جدیدی را از یک تصویر استخراج کنید:

// استخراج رنگ از یک تصویر و افزودن آن به رنگ‌های موجود
makeupApp.addColorFromImage("https://example.com/new-color.jpg", "NEW_CODE", "lips")
    .then(extractedColor => {
        console.log(`رنگ استخراج شده: ${extractedColor}`);
        // اکنون می‌توانید از این رنگ استفاده کنید
    })
    .catch(error => {
        console.error("خطا در استخراج رنگ:", error);
    });
